# Site da Barbearia - Projeto Front-End

Projeto desenvolvido em 2020, durante meus primeiros estudos em programação e desenvolvimento web.

O Site da Barbearia foi criado como uma página institucional, com o objetivo de apresentar uma barbearia, seus serviços e informações principais de forma organizada e visualmente agradável.

Esse projeto representa uma das minhas primeiras experiências práticas com HTML e CSS, aplicando conceitos iniciais de estruturação de páginas, estilização e organização de conteúdo.

---

## Sobre o projeto

A proposta do projeto foi desenvolver um site para uma barbearia, praticando a criação de uma página web estática com foco em apresentação visual.

Durante o desenvolvimento, trabalhei com estrutura HTML, estilização com CSS e organização dos arquivos do projeto, além de utilizar imagens para compor a identidade visual da página.

Por ter sido desenvolvido no início da minha jornada em programação, este projeto tem grande importância no meu aprendizado e registra uma das primeiras etapas da minha evolução como desenvolvedor.

---

## Objetivo

O objetivo principal do projeto foi praticar os fundamentos do desenvolvimento front-end, criando uma página funcional e visualmente organizada.

Com esse projeto, pude aprender melhor sobre:

- Estruturação de páginas com HTML
- Estilização visual com CSS
- Organização de arquivos em um projeto web
- Uso de imagens em páginas HTML
- Criação de páginas institucionais
- Publicação de projetos no GitHub
